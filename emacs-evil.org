#+title: Emacs Evil mode

[[https://github.com/emacs-evil/evil][Evil]] is a Vim emulator for emacs.
Install it with =straight.el=, and turn =evil-mode= on:

#+begin_src emacs-lisp
  (use-package evil
    :config
    (evil-mode))
#+end_src
